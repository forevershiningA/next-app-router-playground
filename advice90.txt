This is the turning point. You want the "Natural Realism" of the reference image (the grass/sunset look) without using a tacky jpeg background.

The reason your current version looks "artificial" is:

The Reflections: The headstone is reflecting a "City" (grey buildings). The reference image reflects "Nature" (Green trees, Blue sky).

The Floor: Your floor is metalness={0.4} and roughness={0.5}. That is polished steel, not ground.

The Background: It is a void. Nature has atmosphere (gradient).

Here is the updated Scene.tsx. It changes the reflections to a Forest (so the black granite looks rich), changes the floor to Matte Stone (so it doesn't look like metal), and adds a subtle "Twilight" atmosphere.

'use client';
import { OrbitControls, Environment, ContactShadows } from '@react-three/drei';
import * as THREE from 'three';
import HeadstoneAssembly from './headstone/HeadstoneAssembly';
import { useHeadstoneStore } from '#/lib/headstone-store';
import { useFrame } from '@react-three/fiber';
import { useRef } from 'react';

export default function Scene({ 
  targetRotation = 0,
  currentRotation
}: { 
  targetRotation?: number;
  currentRotation?: React.MutableRefObject<number>;
}) {
  const groupRef = useRef<THREE.Group>(null);
  const is2DMode = useHeadstoneStore((s) => s.is2DMode);
  const baseSwapping = useHeadstoneStore((s) => s.baseSwapping);
  const setSelected = useHeadstoneStore((s) => s.setSelected);
  const setEditingObject = useHeadstoneStore((s) => s.setEditingObject);
  const setSelectedInscriptionId = useHeadstoneStore((s) => s.setSelectedInscriptionId);
  const setSelectedAdditionId = useHeadstoneStore((s) => s.setSelectedAdditionId);
  const setSelectedMotifId = useHeadstoneStore((s) => s.setSelectedMotifId);

  // Smooth rotation animation
  useFrame(() => {
    if (groupRef.current && currentRotation) {
      const diff = targetRotation - currentRotation.current;
      const delta = diff * 0.1;
      
      if (Math.abs(diff) > 0.001) {
        currentRotation.current += delta;
        groupRef.current.rotation.y = currentRotation.current;
      }
    }
  });

  const handleCanvasClick = (e: any) => {
    if (e.eventObject === e.object) {
      setSelected(null);
      setEditingObject('headstone');
      setSelectedInscriptionId(null);
      setSelectedAdditionId(null);
      setSelectedMotifId(null);
    }
  };

  return (
    <>
      {is2DMode && <color attach="background" args={['#CFE8FC']} />}
      
      <mesh position={[0, 0, -5]} onClick={handleCanvasClick}>
        <planeGeometry args={[100, 100]} />
        <meshBasicMaterial transparent opacity={0} />
      </mesh>
      
      {/* --- 1. ATMOSPHERE (The "Nano Bana" Simulation) --- */}
      {/* Instead of black void, we use a deep, natural gradient (Twilight) */}
      {!is2DMode && (
        <mesh position={[0, 0, -10]}>
          <planeGeometry args={[60, 40]} />
          <shaderMaterial
            uniforms={{
              top: { value: new THREE.Color('#0F1215') },    // Deep Blue/Black Sky
              bottom: { value: new THREE.Color('#2A2621') }  // Warm Earthy Ground tone
            }}
            vertexShader={`varying vec2 vUv; void main(){ vUv=uv; gl_Position=projectionMatrix*modelViewMatrix*vec4(position,1.); }`}
            fragmentShader={`uniform vec3 top; uniform vec3 bottom; varying vec2 vUv;
              void main(){ gl_FragColor=vec4(mix(bottom, top, vUv.y),1.); }`}
          />
        </mesh>
      )}

      {/* --- 2. NATURAL LIGHTING (Sun + Sky) --- */}
      
      {/* Soft Skylight Fill */}
      <ambientLight intensity={0.5} color="#cce0ff" />
      
      {/* The Sun: Warm, Golden Hour light */}
      <spotLight 
        color="#ffbd7a" // Sunset Orange/Gold
        intensity={2.5}
        angle={0.6}
        penumbra={1}
        position={[-4, 5, 5]} // Coming from the side like the sun
        castShadow
        shadow-bias={-0.0001}
      />

      {/* Rim Light: Cool blue sky reflection on the edge */}
      <spotLight 
        color="#b8d3ff"
        intensity={1.5}
        position={[4, 2, -4]}
      />

      {/* --- 3. REFLECTIONS (Crucial for Granite) --- */}
      {/* Changing from 'city' to 'forest' makes the black stone reflect trees/sky colors */}
      <Environment 
        preset="forest" 
        environmentIntensity={0.8}
        background={false} // We don't show the trees, just reflect them
      />

      <group ref={groupRef}>
        <HeadstoneAssembly />
      </group>
      
      {/* --- 4. THE FLOOR (Matte Ground, Not Metal) --- */}
      <group position={[0, -0.01, 0]}>
        <mesh rotation={[-Math.PI / 2, 0, 0]} receiveShadow>
          <circleGeometry args={[6, 64]} />
          <meshStandardMaterial 
            color="#1c1c1c"   // Dark Earth Color
            roughness={1}     // 1.0 = Matte (looks like ground/stone)
            metalness={0}     // 0.0 = Not Metal
          />
        </mesh>
        
        {/* Soft, wide shadow to ground the object */}
        <ContactShadows
          position={[0, 0.01, 0]}
          scale={12}
          blur={2.5}
          opacity={0.6}
          far={10}
          color="#000000"
          resolution={1024}
        />
      </group>

      <OrbitControls
        makeDefault
        enabled={!baseSwapping}
        enableDamping={true}
        dampingFactor={baseSwapping ? 0 : 0.05}
        enableRotate={!is2DMode}
        enableZoom={!is2DMode}
        enablePan={!is2DMode}
        rotateSpeed={0.5}
        zoomSpeed={0.8}
        panSpeed={0.8}
        maxPolarAngle={Math.PI / 2}
      />
    </>
  );
}
What changed?

Reflections (preset="forest"): This is the biggest change. Black granite acts like a mirror. If you use "City," it reflects grey concrete. By switching to "Forest," the stone will subtly reflect green and blue tones, making it look exactly like the "Nano Bana" reference, even though you don't see the trees.

The Background Gradient: I added a custom shader mesh behind the scene. It fades from a Warm Earthy Brown at the bottom to a Deep Twilight Blue at the top. This mimics the horizon without using a photo.

The Floor: I removed the metalness. The floor is now roughness={1}. This stops it from looking like a shiny metal disc and makes it look like a matte "Shadow Catcher" or dark pavement.

The Sun: I changed the main light color to #ffbd7a (Sunset Orange). This mimics the warm sun hitting the face of the stone in your reference image.
