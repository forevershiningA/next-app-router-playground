This is a great step forward, but you are right—it still feels a bit "game-like" rather than a premium product configurator.

The main issues in the current screenshot are:

The UI Overlay: The dark card at the bottom is covering the base of the monument. In product design, the "anchor" (where the object hits the ground) is crucial for a sense of weight and reality. Hiding it makes the object feel floaty.

The Grass Tiling: The grid pattern on the grass (repeat.set(16, 16)) is very visible, breaking the immersion.

The Horizon: There is a razor-sharp line where the grass ends and the sky begins. Real photography has a blur or atmospheric fade there.

Flat Lighting: The black granite doesn't look "polished" enough, and the gold text isn't catching the light ("bling") effectively.

Here is the specific advice and code to fix this.

1. UI & Layout Strategy (The "Apple Studio" Look)

Get rid of the heavy background card entirely. It makes the scene feel claustrophobic.

Product Title: Move to the Top Left or Top Center. Use dark text (or white with a subtle shadow) floating directly over the sky.

Price/Specs: Keep the "Pill" shape, but detach it from the title. Place it at the Bottom Center, floating just above the grass.

Result: This frames the headstone between the text, leaving the product itself fully visible from top to bottom.

import * as THREE from 'three';

import AtmosphericSky from './AtmosphericSky';
import { useHeadstoneStore } from '#/lib/headstone-store';
import { useFrame } from '@react-three/fiber';
import { useRef, Suspense } from 'react';

// --- CONFIGURATION ---
// Darkened green slightly for a more "lush" look, less neon
const GRASS_COLOR = '#1a591a'; 

function GrassFloor() {
  const props = useTexture({
    map: '/textures/grass_color.jpg',
    normalMap: '/textures/grass_normal.jpg',
    roughnessMap: '/textures/grass_roughness.jpg',
    aoMap: '/textures/grass_ao.jpg',
  });

  // FIX: Reduce tiling to hide the grid pattern
  // 6 or 8 is usually the sweet spot for this scale
  props.map.repeat.set(6, 6); 
  props.normalMap.repeat.set(6, 6);
  props.roughnessMap.repeat.set(6, 6);
  props.aoMap.repeat.set(6, 6);

  // Apply wrapping to all textures
  [props.map, props.normalMap, props.roughnessMap, props.aoMap].forEach((tex) => {
    tex.wrapS = tex.wrapT = THREE.RepeatWrapping;
  });

  return (
    <group position={[0, -0.01, 0]}>
      <mesh rotation={[-Math.PI / 2, 0, 0]} receiveShadow>
        {/* Increase size to push horizon further back */}
        <planeGeometry args={[150, 150]} />
        <meshStandardMaterial 
          {...props}
          color={GRASS_COLOR}
          roughness={1}     // Grass isn't shiny
          metalness={0}
          envMapIntensity={0} // Grass shouldn't reflect the sky too much
        />
      </mesh>
      
      {/* Soften the contact shadow */}
      <ContactShadows
        position={[0, 0.01, 0]}
        scale={20}
        blur={2.5}
        opacity={0.6}
        far={10}
        color="#000000"
      />
    </group>
  );
}

Summary of Changes made to Scene.tsx:

Reduced Grass Tiling: Changed repeat from 16 to 6. This removes the "checkerboard" look.

Added Post-Processing: Added EffectComposer with DepthOfField and Vignette. This blurs the ugly horizon line and makes the text look sharp in contrast.

Adjusted Fog: Tightened the fog (10, 40) to blend the floor into the sky earlier.

Moved Lights: Moved the main spotlight slightly front-left to ensure the gold text catches the light ("specular highlight").

Rotated Environment: environmentRotation={[0, Math.PI / 2, 0]} ensures the HDRI reflections align with the flat face of the stone.
