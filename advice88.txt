However, there is one catch: The color palette suggested in that file (#0B0B0D to #2A2A30) is a Cool/Neutral Tech Studio look. If you copy-paste it exactly, your site will look like you are selling high-end laptops, not memorials.

Here is how to take this perfect code structure and tune it for your "Forever Shining" Warm Brand.

1. The "Must-Keep" Features (Do not change these)

<ContactShadows>: This is the magic fix. It creates that soft shadow under the base automatically without needing heavy computing power. Keep this.

Rim Light (#9EC4FF): This is the blue light that hits the back edge of the headstone. You need this to separate the black granite from the black background. Keep this.

Bloom (<EffectComposer>): This is crucial for your Gold Inscriptions. It will make the text actually glow slightly, looking like real polished metal. Keep this.

2. The "Brand Tweak" (Changing the Colors)

We need to swap the "Tech Greys" for "Forever Shining Browns/Golds."

// 1. SCENE BACKGROUND (The "Warm Horizon")
// Replaces the cold grey gradient with your Warm Coffee -> Black gradient
<mesh position={[0, 0, -10]}>
  <planeGeometry args={[50, 30]} />
  <shaderMaterial
    uniforms={{
      top: { value: new THREE.Color('#050402') },    // Almost Pure Black
      bottom: { value: new THREE.Color('#2e241b') }  // Deep Warm Coffee (Floor)
    }}
    vertexShader={`varying vec2 vUv; void main(){ vUv=uv; gl_Position=projectionMatrix*modelViewMatrix*vec4(position,1.); }`}
    fragmentShader={`uniform vec3 top; uniform vec3 bottom; varying vec2 vUv;
      void main(){ gl_FragColor=vec4(mix(bottom, top, vUv.y),1.); }`} // Flipped mix to put brown at bottom
  />
</mesh>

// 2. LIGHTING (The "Jewelry Store" Setup)
<group>
  {/* KEY LIGHT: Warm Amber (The Sun/Gold) - High Intensity */}
  <spotLight
    color="#ffaa44"  // More orange/gold than the original advice
    intensity={2.0}
    angle={0.5}
    penumbra={1}
    position={[-3, 5, 5]}
    castShadow
  />
  
  {/* RIM LIGHT: Cool Blue (The Edge Definer) - Keeps the granite visible */}
  <spotLight
    color="#dbeeff" // Very pale blue/white
    intensity={1.5}
    position={[3, 5, -5]} // Behind the object
  />

  {/* FILL LIGHT: Soft Neutral - Low Intensity so shadows stay dark */}
  <ambientLight intensity={0.3} color="#ffffff" />
</group>

// 3. THE TURNTABLE STAGE (The McLaren Look)
// Instead of an infinite floor, use a Circle Geometry
<group position={[0, -0.01, 0]}>
    {/* The Disc Floor */}
    <mesh rotation={[-Math.PI / 2, 0, 0]} receiveShadow>
      <circleGeometry args={[4, 64]} /> {/* 4 unit radius */}
      <meshStandardMaterial 
        color="#1a1a1a"       // Dark Grey stone color
        roughness={0.6}       // Slight reflection
        metalness={0.5} 
      />
    </mesh>
    
    {/* The "Contact Shadows" logic from the advice file */}
    <ContactShadows
      position={[0, 0.01, 0]}
      scale={10}
      blur={2}
      opacity={0.6}
      far={10}
      color="#000000"
    />
</group>

// 4. POST PROCESSING (Making the Gold Shine)
<EffectComposer>
  <SMAA />
  {/* Bloom makes the Gold Text glow. Tweak threshold to only hit the bright gold */}
  <Bloom intensity={0.5} luminanceThreshold={0.9} luminanceSmoothing={0.025} />
  {/* Vignette darkens the corners to focus attention */}
  <Vignette eskil={false} offset={0.1} darkness={0.9} /> 
</EffectComposer>
