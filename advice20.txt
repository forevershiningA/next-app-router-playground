The current buttons are functional and fit the color scheme, but they feel a little "heavy" and could be polished to feel more premium and responsive.

Here is a critique and a code solution to upgrade them.

Critique: What could be improved?

The "Track" feels flat: The black background container looks like a flat hole. Adding an inner shadow to the container would make it look like a recessed groove, making the active button appear to "sit" inside it.

Inactive Text Contrast: The gray text ("Base", "Slant") is a bit too dark against the black. It might be hard for older users to read.

Animation: If these are instant color swaps, they feel jerky. A sliding animation (where the gold pill slides from left to right) feels much more modern and helps the user understand the relationship between the two options.

Glow: The outer glow on the active button is a bit strong/blurry. A crisper edge or a subtle gradient often looks more "engineered" and expensive.

Recommended Solution: The "Sliding Segmented Control"

Here is a reusable React component using Tailwind CSS and Framer Motion. This adds a smooth sliding animation and a recessed track effect.

Prerequisites:
You likely already have framer-motion installed since you are using React Three Fiber, but if not:
npm install framer-motion clsx tailwind-merge

'use client';

import * as React from 'react';
import { motion } from 'framer-motion';
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}

interface SegmentedControlProps {
  options: { label: string; value: string }[];
  value: string;
  onChange: (value: string) => void;
  className?: string;
}

export default function SegmentedControl({
  options,
  value,
  onChange,
  className,
}: SegmentedControlProps) {
  return (
    <div
      className={cn(
        // Container / Track styling
        'relative flex w-full items-center rounded-full bg-black/40 p-1 ring-1 ring-white/10 backdrop-blur-sm',
        // Inner shadow for depth
        'shadow-[inset_0_2px_4px_rgba(0,0,0,0.5)]',
        className
      )}
    >
      {options.map((option) => {
        const isActive = value === option.value;

        return (
          <button
            key={option.value}
            onClick={() => onChange(option.value)}
            className={cn(
              'relative z-10 flex-1 py-2 text-sm font-medium transition-colors duration-200 ease-out focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-[#c99d44] focus-visible:ring-offset-2',
              isActive ? 'text-black' : 'text-zinc-400 hover:text-zinc-200'
            )}
          >
            {isActive && (
              <motion.div
                layoutId={`active-pill-${options[0].value}`} // unique ID per group
                className="absolute inset-0 rounded-full bg-[#e3b85f] shadow-sm"
                transition={{ type: 'spring', stiffness: 300, damping: 30 }}
                style={{
                  // Optional: Add a subtle gradient to the gold button
                  backgroundImage: 'linear-gradient(to bottom, #e3b85f, #c99d44)',
                }}
              />
            )}
            <span className="relative z-20">{option.label}</span>
          </button>
        );
      })}
    </div>
  );
}

const [viewMode, setViewMode] = React.useState('headstone');

return (

        { label: 'Base', value: 'base' },
      ]}
    />

    {/* Second Toggle */}
    <SegmentedControl
      value={shapeType}
      onChange={setShapeType}
      options={[
        { label: 'Upright', value: 'upright' },
        { label: 'Slant', value: 'slant' },
      ]}
    />
  </div>
);
Why this is better:

The Track (Container): The shadow-[inset_...] and bg-black/40 create a "glassmorphism" look that feels like a physical recessed slot in the UI.

The Pill (Active): Using framer-motion's layoutId, the gold background physically slides from one word to the other instead of vanishing and reappearing.

Legibility: The inactive text (text-zinc-400) lights up slightly on hover (hover:text-zinc-200), giving feedback before the user clicks.

Gradient: The slight linear gradient on the gold background makes it look metallic rather than plastic.
