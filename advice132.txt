The issue is that the <ContactShadows /> component is placed inside the rotating group. This acts like a "Lazy Susan" or turntable: when the headstone rotates, the invisible "floor" that catches the shadow rotates with it, making the shadow appear to spin physically rather than staying projected on the ground.

To fix this, you need to move <ContactShadows /> outside the groupRef so it stays static in the world while the headstone rotates above it.

Here is the corrected SceneContent component:

const SceneContent = ({ targetRotation }: { targetRotation: number }) => {
  const groupRef = useRef<THREE.Group>(null);
  
  // Thickness/2 (Front Face) + Bevel Thickness + Gap
  const textZ = (STONE_THICKNESS / 2) + BEVEL_SIZE + 0.02;

  useFrame((state, delta) => {
    if (groupRef.current) {
      const step = 0.1;
      const diff = targetRotation - groupRef.current.rotation.y;
      
      if (Math.abs(diff) > 0.001) {
        groupRef.current.rotation.y += diff * step;
      }
      
      groupRef.current.rotation.y += Math.sin(state.clock.elapsedTime * 0.5) * 0.0005;
    }
  });

  return (
    <>
      {/* 1. Rotating Group: Contains only the physical objects */}
      <group ref={groupRef} position={[0, -1.35, 0]}>
        <SerpentineHeadstone width={STONE_WIDTH} height={STONE_HEIGHT} thickness={STONE_THICKNESS} />
        <Base stoneWidth={STONE_WIDTH} />
        
        <group position={[0, 0, 0]}>
          <GoldText
            position={[0, BASE_HEIGHT + STONE_HEIGHT * 0.60, textZ]}
            fontSize={0.11}
            text="In Loving Memory"
            font="/fonts/Garamond.ttf"
          />
          
          <GoldText
            position={[0, BASE_HEIGHT + STONE_HEIGHT * 0.50, textZ]}
            fontSize={0.22}
            text="Eleanor Rose"
            font="/fonts/Garamond.ttf"
            fontWeight="bold"
          />
          
          <GoldText
            position={[0, BASE_HEIGHT + STONE_HEIGHT * 0.40, textZ]}
            fontSize={0.11}
            text="☼ 1945   ✟ 2023"
            font="/fonts/Garamond.ttf"
          />
        </group>
      </group>

      {/* 2. Static Shadow: Moved OUTSIDE the rotating group */}
      {/* Positioned at Y=-1.35 to match the bottom of the group */}
      <ContactShadows 
        position={[0, -1.35, 0]}
        opacity={0.4} 
        scale={10} 
        blur={2} 
        far={3} 
        resolution={512} 
        color="#000000"
      />
    </>
  );
};
Key Changes:

Moved Component: <ContactShadows /> was moved out of the <group ref={groupRef}>.

Updated Position: The position of the shadow is now [0, -1.35, 0]. This aligns it with the global Y position of the headstone group (so the shadow appears at the feet of the monument), but ensures the shadow plane itself does not rotate.
