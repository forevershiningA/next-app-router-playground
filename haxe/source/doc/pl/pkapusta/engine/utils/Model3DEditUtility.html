@:value({ forceErase : false, tryPlaceChildsOnParent : true })staticerase(model:IModel3D, tryPlaceChildsOnParent:Bool = true, forceErase:Bool = false):Int

Removes the model from the scene while performing additional operations, i.e. inserting children in place of the parent (if possible), checking whether it can be safely removed.

Parameters:

model

model3d instance

tryPlaceChildsOnParent

If true, it tries to put children in place of the parent

forceErase

Forces deletion even if it is not a safe deletion (e.g. if children cannot be replaced by parent)

Returns:

Execution code: -1: FAULT, 0: SUCCESS, 1: CANT ERACE CHILDS

staticeraseEnabled(model:IModel3D):Bool

Returns whether 3d model removal is enabled

Parameters:

model

model3d instance

Returns:

true or false

@:value({ cloneProperties : false, forceReplace : false })staticreplace(currentModel:IModel3D, newModel:IModel3D, forceReplace:Bool = false, cloneProperties:Bool = false):Int

It replaces the model with another one that can replace it in the same place, e.g. inscriptions on a board above other inscriptions or a motif. It tries to set the same childs and properties i.e. size and position if the new model supports them.

Parameters:

currentModel

current model3d instance

newModel

new model3d instance

forceReplace

forces replacement even if inconsistencies are detected between one model and another

cloneProperties

if true, it also clones the properties

Returns:

Execution code: -1: FAULT, 0: SUCCESS, 3: CANT REPLACE WITH ALL CHILDS
