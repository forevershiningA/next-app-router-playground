Engine3D

The root Engine 3D object that initializes the model, view and controller.

Engine3DController

Main controller. Manages the 3d project state, initializes a new project, loads and saves.

Engine3DModel

The main model that controls the state of the project. It is not a Model3D object that manages the state of individual models. The Model3D object is located here: pl.pkapusta.engine.model.Model3D. This object implements the behavior of the project, but you should not perform any salaried operations on it. The Engine3DController object is used to control the state of the project from the user's side (located here: pl.pkapusta.engine.Engine3DController).

Engine3DView

Main 3D View and camera control for Engine 3D. Rendering a scene to an image.

IEngine3D

The root Engine 3D interface for acces to the model, view and controller.

IEngine3DController

Main controller interface. Manages the 3d project state, initializes a new project, loads and saves.

IEngine3DView

Main 3D View interface and camera control for Engine 3D. Rendering a scene to an image.
