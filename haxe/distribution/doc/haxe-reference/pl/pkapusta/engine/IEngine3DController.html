interface IEngine3DController

package pl.pkapusta.engine

extends IEventDispatcher

extended by Engine3DController

Main controller interface. Manages the 3d project state, initializes a new project, loads and saves.

Methods

closeProject():Void

Closes the current project

closeProjectAndDisposeChildren():Void

Closes the current project and clears all models and their resources

getCurrentProject():IProject3D

Get current project instance

@:value({ onComplete : null, context : null })loadProject(urlOrData:Dynamic, ?context:ProjectContext, ?onComplete:IProject3D ‑> Void):Void

Loads project from file or from byteArray. Current project will be destoyed.

Parameters:

urlOrData

Path to project file (String or URLRequest) or project binary data (ByteArray)

context

Specify load context data and customize loading operation (for example: relative path for external M3D model, parsing callbacks, etc.)

onComplete

Callbacks when loading is done

newEmptyProject():Void

Creates new empty engine project. Current project will be destoyed.

@:value({ output : "Uint8Array", customizator : null })saveProject(?customizator:ProjectSaveCustomizator, output:String = "Uint8Array"):Dynamic

Saves current project do binary data. After saving project is still exists.

Parameters:

customizator

Put customizator class here for customize save data. Using this class you can affect to saving process.

output

Output type, possible values from Engine3D.enums.IOFormat: Uint8Array, Base64String, Bytes

Returns:

Binary data for project.
