# Quick wins (least effort → most impact)

* **Neutral studio gradient**: fade top→bottom (near-black to warm grey) so gold text reads cleanly.
* **Soft ground plane**: a large, slightly rough plane with a **contact shadow** under the stone (no sharp horizon line).
* **Rim + key light**: warm key (gold accents), cool rim (separates from background), low-intensity fill.
* **Subtle fog/vignette**: center focus on the memorial, hide edges of the canvas.
* **Low-gloss ground**: micro-roughness so the base doesn’t mirror; just a faint reflection.
* **HDRI with low intensity**: use a neutral studio HDRI, then drive your own lights; keep envMapIntensity ~0.2–0.5.

# Palette suggestions

* Background top → bottom: `#0B0B0D → #2A2A30`
* Ground: `#3A3736` rough stone, roughness 0.9, metalness 0.0
* Lights: key color `#FFD08A` (warm), rim `#9EC4FF` (cool)

# R3F drop-in (studio look)

```tsx
// Scene background
useEffect(() => {
  gl.setClearColor('#0B0B0D', 1);
}, [gl]);

// Gradient with a big quad behind everything
<mesh position={[0, 0, -10]}>
  <planeGeometry args={[50, 30]} />
  <shaderMaterial
    uniforms={{
      top: { value: new THREE.Color('#0B0B0D') },
      bottom: { value: new THREE.Color('#2A2A30') }
    }}
    vertexShader={`varying vec2 vUv; void main(){ vUv=uv; gl_Position=projectionMatrix*modelViewMatrix*vec4(position,1.); }`}
    fragmentShader={`uniform vec3 top; uniform vec3 bottom; varying vec2 vUv;
      void main(){ gl_FragColor=vec4(mix(top,bottom, vUv.y),1.); }`}
  />
</mesh>

// Environment kept subtle
<Environment preset="studio" environmentIntensity={0.35} />

// Lights
<group>
  {/* warm key, slightly above/front-left */}
  <spotLight
    color="#FFD08A"
    intensity={1.25}
    angle={0.45}
    penumbra={0.8}
    position={[-2.2, 3.0, 4.0]}
    castShadow
  />
  {/* cool rim from back-right */}
  <directionalLight
    color="#9EC4FF"
    intensity={0.6}
    position={[3.0, 2.5, -2.5]}
  />
  {/* gentle fill */}
  <hemisphereLight intensity={0.25} color="#FFFFFF" groundColor="#303035" />
</group>

// Ground plane + contact shadow
<mesh rotation={[-Math.PI / 2, 0, 0]} position={[0, -0.01, 0]} receiveShadow>
  <planeGeometry args={[50, 50]} />
  <meshStandardMaterial color="#3A3736" roughness={0.9} metalness={0.0} />
</mesh>
<ContactShadows
  position={[0, 0, 0]}
  scale={8}
  blur={2.5}
  opacity={0.4}
  far={8}
/>

// Subtle atmospheric focus
<fog attach="fog" args={['#0B0B0D', 12, 30]} />
```

# Optional post-processing

Use `@react-three/postprocessing`:

* **Bloom** (very low threshold) to lift gold inscription highlights.
* **Vignette** (mild) to darken corners.
* **SMAA** for clean edges.

```tsx
<EffectComposer>
  <SMAA />
  <Bloom intensity={0.2} luminanceThreshold={0.8} luminanceSmoothing={0.1} />
  <Vignette eskil={false} offset={0.4} darkness={0.6} />
</EffectComposer>
```

# Variants you can switch to

1. **Outdoor memorial-garden**: HDRI “dawn”/“park” with envIntensity ~0.2, soft sky gradient `#0E1A22 → #6B7D8A`, very long, soft contact shadow, a hint of greenish ground tint.
2. **Showroom**: gentle checker concrete (`roughness 0.8`), cooler key light, desaturated background `#141414 → #242424`.

# Accessibility & performance

* Keep contrast ratio > 4.5:1 between inscriptions and background; the studio gradient above does this.
* Cap total post-FX to keep 60 fps; keep shadow map at 1024–2048, use `PCFSoftShadowMap`.
