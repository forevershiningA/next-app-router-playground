## Kontekst

Mamy model inflacji ASG (Asymptotically Safe Gravity) w ramce Jordana:
- Działanie: `S = ∫√-g [F(χ)R - ½(∂χ)² - V(χ)]`
- `F(χ) = M_Pl²[1 + β·exp(-(χ-χ₀)²/Δ²)]`, `V(χ) = Λ⁴[1-exp(-χ/μ)]²`
- 4 wolne parametry ASG: β, Δ, χ₀, μ

Poprzedni run v13 (`chains/asg_polychord_v13/`) skończył się z:
- `ln(Z) = -1409.25 ± 0.48`
- Tylko **261 równowagowych próbek posterioru** (za mało — n_live=50 w 29D)
- Słaba zbieżność MCMC: R-1 > 0.4 dla β, Δ, χ₀
- Zbyt szeroki prior ASG (χ₀, μ marnowały ~92% objętości)

## Dlaczego v14 jest lepszy (skrót)

| | v13 | v14 | Efekt |
|---|---|---|---|
| nlive | 50 | **100** | 2× więcej próbek |
| num_repeats | 58 | **29** | Ten sam koszt (100×29 = 50×58 = 2900) |
| grade_frac | [2,5] | **[1,8]** | ~75% mniej wywołań CLASS (bottleneck) |
| precision | 0.01 | **0.005** | +10% ndead gratis |
| χ₀ prior | [1.5, 9.0] | **[2.8, 6.5]** | 12× mniejsza objętość prioru |
| μ prior | [0.5, 10.0] | **[3.0, 6.1]** | ↑ (łącznie z χ₀) |

Oczekiwane rezultaty: ~500 próbek, ln(Z) ~−1407, błąd ln(Z) ~0.34

## Jeśli coś nie działa

1. **Brak pliku .param** → sprawdź git pull z repo Windows
2. **CLASS nie startuje** → sprawdź `clik_profile.sh` i `conda activate plc311`
3. **PolyChord crash** → sprawdź logi, prawdopodobnie brakuje `prior_volume` w likelihood bridge
4. **grade_frac nie przyjmuje "1 8"** → spróbuj bez cudzyslowów: `--PC_grade_frac 1 8` lub sprawdź składnię MontePython dla tej wersji
