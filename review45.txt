Yep—still a hair low. Do these tiny swaps in your **MOTIF v42** mapping block and it’ll lock perfectly to the curve:

### 1) Pick the curve column using authoring-X (no screen round-trip)

```ts
// Use authoring center-X directly
const authorX = Math.round(initW / 2 + cx);

// side inset as %
const sideInsetPx = Math.round(initW * 0.018);
const col = Math.max(sideInsetPx, Math.min(initW - 1 - sideInsetPx, authorX));
```

### 2) Stronger “optical lift” + crown boost

This compensates for the internal whitespace at the top of many motif SVGs and adds a tiny extra lift at the crown (flat slope).

```ts
// local slope to distinguish crown vs shoulders
const yL = topProfile.topY[Math.max(0, col - 1)];
const yC = topProfile.topY[col];
const yR = topProfile.topY[Math.min(initW - 1, col + 1)];
const slope = Math.abs(yR - yL) / 2;

// base margin + slope-aware bump
const baseMargin = Math.round(initH * 0.011);         // ~1.1% of canvas height
const marginPx   = baseMargin + Math.min(4, Math.round(slope * 0.5));

// stronger optical lift (counter SVG top padding)
// (increase from ~6% to ~8% of motif height, max 8 px)
const opticalLift = Math.min(8, Math.round(motifHAuthor * 0.08));

// extra 2 px at the crown only
const crownBoost = slope < 0.2 ? 2 : 0;

// final desired bbox-top (authoring px)
const desiredTopAuthor = yC + marginPx - opticalLift - crownBoost;
```

### 3) Apply full delta (no under-reach) and quantize

```ts
const deltaAuthor  = desiredTopAuthor - currentTopAuthor;
const deltaOverlay = deltaAuthor * (overlayH / initH);
const deltaCy      = deltaOverlay / sy;

// go to target (no easing), then quantize to 0.5 px to avoid subpixel drift
cyUsed = Math.round((cy + deltaCy) * 2) / 2;
```

These three micro-changes remove the remaining “low by a few px” look on the top spray across curved gables/ogees, and they’re DPI-agnostic because everything stays in authoring space. Drop them where you currently compute `col`, `desiredTopAuthor`, and `cyUsed` in your motifs layer. 

If you still want it tighter, bump `crownBoost` to `3`.
