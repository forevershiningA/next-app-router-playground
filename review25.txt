### 1) Ratio is being applied to the *displayed* SVG size, not the motif’s **intrinsic** (viewBox) size

When `height` isn’t saved, your code builds `hCanvas` from `svgDims.height` and then multiplies by `uniformScale` again → double-scale and drift at the arch. Fix by using the **intrinsic** SVG dimensions (viewBox), not the rendered box, then scale **once**:

```ts
// cache by src → {vw, vh}
async function getIntrinsicDims(src: string): Promise<{vw:number; vh:number}> {
  if (intrinsic[src]) return intrinsic[src];
  const text = await (await fetch(src)).text();
  const m = text.match(/viewBox\s*=\s*"([\d.\s-]+)"/i);
  if (!m) return intrinsic[src] = { vw: 100, vh: 100 };
  const [, vb] = m;
  const [, , vw, vh] = vb.trim().split(/\s+/).map(Number);
  return intrinsic[src] = { vw, vh };
}

// sizing (ratio path)
const { vw, vh } = await getIntrinsicDims(motifSrc);   // authoring units
const ratio = Number(motif.ratio ?? 1);                // dimensionless
const hCanvas = vh * ratio;                            // authoring space
const wCanvas = vw * ratio;                            // authoring space

// final display size (scale ONCE)
const heightPx = hCanvas * uniformScale;
const widthPx  = wCanvas  * uniformScale;
```

Do **not** use `svgDims.height` (a DOM/display measure) in the ratio branch. That’s what keeps shrinking the top leaves/birds.

### 2) Normalize DPR for positions **and** pixel heights only when those values were saved in pixels

Your legacy saved `x,y,font_size,height,width` in **physical px**; `ratio` is **already normalized** (dimensionless). So:

```ts
const usesPhysical = scalingFactors.usesPhysicalCoords;
const dpr = scalingFactors.designDpr ?? 1;

// positions
const xCanvas = usesPhysical ? (motif.x ?? 0) / dpr : (motif.x ?? 0);
const yCanvas = usesPhysical ? (motif.y ?? 0) / dpr : (motif.y ?? 0);

// explicit px size (if present)
const pxHCanvas = motif.height != null
  ? (usesPhysical ? motif.height / dpr : motif.height)
  : null;

// choose size path
const hCanvas = pxHCanvas ?? (vh * ratio);
const wCanvas = pxHCanvas ? (pxHCanvas * (vw / vh)) : (vw * ratio);
```

Then position with the **same** scale as inscriptions:

```ts
const left = offsetX + (xCanvas + initW/2) * uniformScale;
const top  = offsetY + (yCanvas + initH/2) * uniformScale;
// style: transform: 'translate(-50%, -50%)'
```
